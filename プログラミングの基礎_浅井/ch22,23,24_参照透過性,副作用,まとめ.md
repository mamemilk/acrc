## 22.1

2つの方法を考えてみた。
- -1始まりとして、インクリメントを先にやる。
- 文字列を返してからインクリメントする。 

アプローチ1
```ocaml
let count = ref (- 1) ;;
let gensym moji = (count := !count + 1; moji ^ string_of_int !count );;

gensym "a";;
gensym "a";;
gensym "x";;
```

アプローチ2
```ocaml
let count = ref 0 ;;
let gensym moji = let ans = moji ^ string_of_int !count in (count := !count + 1; ans);;

gensym "a";;
gensym "a";;
gensym "x";;
```
