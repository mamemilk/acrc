# TCP/IPとは

## 1章 : 前説

TCP : Transmission Control Protocol
IP : Internet Protocol

OSI参照モデル
- アプリケーション層
- プレゼンテーション層
- セッション層
- トランスポート層
- ネットワーク層
- データリンク層
- 物理層


実際のTCP/IPでの階層定義は、
- アプリケーション層
- トランスポート層
- インターネット層
- リンク層


TCP/IPは、IETF : Internet Engineering Task Forceが標準化団体
物理層とデータリンク層はIEEEが決めている。
