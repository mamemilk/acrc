# 5章：原始帰納的関数

## 5.1

かけ算が、原始帰納的であることを示す。

```
# 足し算の場合
f(x,y) = x + y
-> 
f(x,0) = x,
f(x,y+1) = suc(f(x,y))


# かけ算の場合 
f(x,y) = x * y
-> 
f(x,0) = 0
f(x,y+1) = f(x,y) + x

足し算が原始帰納的なことは上記で示されているため、上記により、かけ算も原始帰納的であることが示される。
```
